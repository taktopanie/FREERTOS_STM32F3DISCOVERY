**Projects utilizing FreeRTOS library**
#
**1. GPS_ublox** - uses 2 UARTS to communicate between GPS module and USER
  *  UART2 is used for **GPS module**
        *  PA2 - TX
      	*  PA3 - RX
  *  UART3 is used for **user data**
       *  PB10 - TX
       *  PB11 - RX
